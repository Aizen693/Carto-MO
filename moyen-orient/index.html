Selectionner une periode
Selectionnez une periode en bas (2005-2006, 2007-2008...) ou utilisez le slider. Le bouton Play anime toutes les periodes automatiquement.
◉
Les points
Chaque point represente un evenement. La couleur indique l'acteur, la taille indique le nombre de victimes. Cliquez pour voir le detail complet.
◇
Cotation du renseignement
Chaque evenement est evalue selon la doctrine OTAN. La lettre indique la fiabilite de la source (A = totalement fiable → F = fiabilite non estimable). Le chiffre indique la veracite du contenu (1 = corroboree par d'autres sources → 6 = veracite non estimable).A1 = totalement fiable / corrobore · B2 = habituellement fiable / probablement vrai · C3 = assez fiable / peut-etre vrai · F6 = non estimable
